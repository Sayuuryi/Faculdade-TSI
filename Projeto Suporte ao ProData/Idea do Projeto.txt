Nome do Projeto: 
Sistema de Ajuda para o Sistema de Requisições de Compras Hospitalares

Objetivo do sistema é resolver os problemas:

Funcionários da prefeitura utilizam um sistema para realizar requisições de compras de 
materiais hospitalares, como gaze, cadeiras de rodas e medicamentos.

Muitas vezes os usuários cometem erros simples durante o uso do sistema e precisam entrar 
em contato com o suporte de TI para resolver problemas que poderiam ser solucionados de 
forma rápida com orientações simples.

Solução dos problemas:

Desenvolvimento de um site de ajuda com:

- guias passo a passo
- explicação de erros comuns
- perguntas frequentes
- contato com suporte

O objetivo é reduzir chamadas para o suporte de TI e facilitar o uso do sistema para os
funcionários.

- Usuários do sistema
- Funcionários da prefeitura
- Setor administrativo

- Funcionários do hospital // Não tenho certeza se isso vai.
