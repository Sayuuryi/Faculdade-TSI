- Questões discursivas:

6. Como o array tamanhos é preenchido e qual é sua utilidade?


Resposta: O array tamanhos é preenchido usando um loop for que percorre cada palavra do array palavras. Para cada palavra, o comprimento da palavra (palavras[i].length) é adicionado ao array usando Array.push. Isso permite saber quantas letras cada palavra possui.


7. Explique o que faz a expressão ${palavras.join(", ")} no código. Qual é a utilidade desse comando e como ele transforma o array palavras para exibição no HTML?


Resposta: A expressão ${palavras.join(", ")} pega todos os elementos do array palavras e os transforma em uma única string, separando cada palavra por uma vírgula e um espaço. Isso permite que o array seja exibido de forma legível no HTML, mostrando todas as palavras da frase digitada pelo usuário em sequência. É uma forma prática de apresentar os dados de um array para o usuário de maneira organizada e compreensível.
